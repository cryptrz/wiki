On the Midland XT70, the full PMR446 channel list is:

| Ch | Frequency (MHz) |
|----|-----------------|
| P1 | 446.00625 |
| P2 | 446.01875 |
| P3 | 446.03125 |
| P4 | 446.04375 |
| P5 | 446.05625 |
| P6 | 446.06875 |
| P7 | 446.08125 |
| P8 | 446.09375 |

These are the 8 base PMR446 frequencies, license-free to use. The other 16 PMR channels on the XT70 are the same 8 frequencies pre-programmed with CTCSS tones.
